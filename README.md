<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Pymatgen（Python 材料基因组学）是一个强大的开源 Python 材料分析库。以下是一些主要功能：</font></font></p>
<ol dir="auto">
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">用于表示</font></font><code>Element</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">、</font></font><code>Site</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">和对象</font></font><code>Molecule</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">的高度灵活的类</font></font><code>Structure</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">。</font></font></li>
<li><font style="vertical-align: inherit;"></font><a href="https://cms.mpi.univie.ac.at/vasp" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">广泛的输入/输出支持，包括对VASP</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">、</font></font><a href="https://abinit.org" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">ABINIT</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">、</font></font><a href="https://wikipedia.org/wiki/Crystallographic_Information_File" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">CIF</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">、</font></font><a href="https://gaussian.com" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Gaussian</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">、</font></font><a href="https://wikipedia.org/wiki/XYZ_file_format" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">XYZ</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">和许多其他文件格式的</font><font style="vertical-align: inherit;">支持。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">强大的分析工具，包括生成相图、普贝图、扩散分析、反应等。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">电子结构分析，例如态密度和能带结构。</font></font></li>
<li><font style="vertical-align: inherit;"></font><a href="https://materialsproject.org" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">与Materials Project</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> REST API集成</font><font style="vertical-align: inherit;">。</font></font></li>
</ol>

<ol dir="auto">
<li><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">它（相当）稳健。</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> Pymatgen 被数千名研究人员使用，是为</font></font><a href="https://materialsproject.org" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">材料项目</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">提供支持的分析代码。它生成的分析每天都经过严格审查。错误往往会很快被发现并纠正。 Pymatgen 还使用 Github Actions 进行持续集成，这确保每个新代码都通过一套全面的单元测试。</font></font></li>
<li><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">这是有据可查的。</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">我们已经编写了相当全面的文档来帮助您快速掌握它。</font></font></li>
<li><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">它是开放的。</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">您可以自由使用和贡献</font></font><code>pymatgen</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">。这也意味着</font></font><code>pymatgen</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">正在不断改进。我们会将您贡献的任何代码归因于您指定的任何出版物。做出贡献</font></font><code>pymatgen</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">意味着您的研究变得更加明显，这意味着更大的影响力。</font></font></li>
<li><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">它很快。</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">中的许多核心数值方法</font></font><code>pymatgen</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">已通过</font></font><code>numpy</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">/中的矢量化进行了优化</font></font><code>scipy</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">。这意味着坐标操作速度很快。 Pymatgen 还附带一个完整的系统来处理周期性边界条件。</font></font></li>
<li><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">它将在附近。</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> Pymatgen 不是一个宠物研究项目。它用于成熟的材料项目。</font></font><a href="https://materialsvirtuallab.org" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">材料虚拟实验室</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">、ABINIT 小组和许多其他研究小组</font><font style="vertical-align: inherit;">也正在积极开发和维护它。</font></font></li>
<li><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">不断发展的开发者和附加组件生态系统</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">。 Pymatgen 得到了世界各地材料科学家的贡献。我们现在还拥有一个支持附加组件的架构，可以</font></font><code>pymatgen</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">进一步扩展 的功能。查看</font></font><a href="https://pymatgen.org/contributing" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">贡献页面</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">和</font></font><a href="https://pymatgen.org/addons" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">附加组件页面</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">以获取详细信息和示例。</font></font></li>
</ol>
